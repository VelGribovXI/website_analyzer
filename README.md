# 📊 Website Traffic Analyzer

## 🎯 Что это?

**Website Traffic Analyzer** - это мощный веб-инструмент для анализа посещаемости веб-сайтов. Просто введите URL, и получите наглядную статистику с красивыми графиками!

## ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| 📈 **График посещаемости** | Визуализация ежедневной динамики за 30 дней |
| 📱 **Анализ устройств** | Распределение трафика по типам устройств |
| 🔍 **Быстрый анализ** | Анализ любого сайта за 2 секунды |
| 🎨 **Современный UI** | Адаптивный дизайн с темной градиентной темой |
| 📊 **Интерактивные графики** | Взаимодействие с данными через Chart.js |
| 🚀 **Высокая производительность** | Быстрая загрузка и отзывчивый интерфейс |

## 📊 Что вы увидите?

### 1. Общая статистика
- Общее количество посещений за период
- Анализируемый URL
- Тип вашего устройства

### 2. Динамика посещений
- Линейный график с ежедневными показателями
- Тренды роста или падения
- Пиковые дни посещаемости

### 3. Устройства пользователей
- Круговая диаграмма распределения
- Процентное соотношение
- Количество посетителей с каждого типа устройств

❓ Частые вопросы

Вопрос: Работает ли с реальными сайтами?
Ответ: Да, но для получения реальной статистики нужно подключить API аналитики (GA4, Яндекс.Метрика). Сейчас используется демо-режим.

Вопрос: Можно ли использовать в коммерческих целях?
Ответ: Да, лицензия MIT позволяет коммерческое использование.

Вопрос: Как добавить свой источник данных?
Ответ: Смотрите документацию в папке /docs, там есть примеры интеграции.
